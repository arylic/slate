## Disable Multiroom Mode

Turns off Multiroom mode on a host player, and resolves the entire group.

Sub-Command: `Ungroup`

> Request format:

```html
GET /httpapi.asp?command=multiroom:Ungroup
```

> Example Response:

```plaintext
OK
```

<aside class="notice">
The response isn't in JSON format. It is just plaintext.
</aside>
